\doxysubsection{Description détaillée}
Permet de résoudre un système linéaire de type \$\+Ax = b\$ avec la méthode itérative de Gauss-\/\+Seidel. 

\begin{DoxyNote}{Note}
La matrice A doit être strictement diagonale dominante pour garantir la convergence. Les éléments de la diagonale principale doivent être non nuls. 
\end{DoxyNote}
